# Proyecto: Simulador de Venta de Pasajes Interurbanos

Este es un proyecto universitario desarrollado en Python que simula la venta de pasajes para una línea de buses.

## Descripción

El programa ofrece al usuario un menú de destinos (Santiago, Arica, Temuco, Concepción) con sus respectivos precios. El usuario puede seleccionar un destino, indicar si desea pasaje de ida y vuelta (lo que duplica el costo) e informar la cantidad de equipaje.

El sistema calcula el costo total, aplicando un cobro adicional por cada maleta extra sobre el límite permitido. Al final, muestra un resumen de la venta y mantiene un conteo de las ventas por ciudad.

## Funcionalidades Principales

* Menú de selección de destinos.
* Lógica de precios para pasajes de ida vs. ida y vuelta.
* Cálculo de cobros adicionales por exceso de equipaje.
* Emisión de un boleto simulado con el desglose del total.
* Contador de ventas por cada destino.

## Tecnologías Utilizadas

* **Python:**
    * Estructuras de datos (Listas, Diccionarios).
    * Bucles `while` y condicionales `if/else`.
    * Manejo de entrada (input) del usuario.
